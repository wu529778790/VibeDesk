# VibeDesk Roadmap

[中文版本](#中文版本)

## English Version

### v0.2.x - Stability & Bug Collection

**Bug Collection**

- [ ] Sentry integration (crash reporting + performance monitoring)
- [ ] Anonymous usage analytics
- [ ] Auto-submit crash logs (no manual user feedback needed)

**Stability**

- [ ] TURN server integration (NAT traversal)
- [ ] Auto-reconnect mechanism
- [ ] Connection quality indicator (latency, packet loss)
- [ ] ICE connection timeout handling

**Bug Fixes**

- [ ] Coordinate scaling (client widget → host screen)
- [ ] macOS input injection (CGEvent)

### v0.3.x - Platform Expansion & Features

**Platform Expansion**

- [ ] Windows ARM64 support
- [ ] Windows 7/8 compatibility
- [ ] macOS Apple Silicon optimization
- [ ] Linux input injection (xdotool/uinput)

**Feature Enhancement**

- [ ] File transfer
- [ ] Clipboard sync (bidirectional)
- [ ] Multi-monitor support
- [ ] Session recording
- [ ] Audio streaming

### v1.0+ - New Platforms & Enterprise

**New Platforms**

- [ ] Android (host mode)
- [ ] iOS (host mode)
- [ ] Web browser client

**Enterprise Features**

- [ ] Password protection for rooms
- [ ] Session audit logs
- [ ] Role-based access control
- [ ] Self-hosted signal server authentication
