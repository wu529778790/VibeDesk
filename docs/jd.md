# 招聘 JD

## 关于我们

我们是一支以 AI驱动开发（Vibe Coding）为核心的产品团队，专注于移动端、Web 平台及 AI Agent 工具链的研发。我们打破传统开发模式，将 AI 深度融入日常研发流程，探索工程师与 AI 协同的全新范式，推动前端与 AI 的深度融合，打造卓越的产品体验。

## 核心职责

- 负责 Web、移动端应用核心开发，构建高质量、可复用的前端架构与组件体系
- 负责前端状态管理与样式系统，确保数据流清晰、界面风格统一
- 集成复杂交互组件，保障良好的用户体验
- 参与前端应用相关开发，处理 Canvas 渲染、触控交互及动画效果
- 编写测试用例与组件文档，保障代码质量与可维护性
- 对接 LLM API，参与 AI 对话界面及 Agent 工作流的前端实现

## 核心技术要求

- 精通 TypeScript、主流前端框架及构建工具，具备 monorepo 架构实践经验
- 熟练掌握前端状态管理与样式解决方案，能独立搭建可复用的设计系统
- 拥有移动端跨平台开发经验，能实现原生级体验
- 具备良好的工程素养，重视代码质量、错误处理、日志规范
- 理解 Feature-First 架构及 DDD 分层思想，能实现全链路类型安全
- 深度运用 AI 编程工具，高效驾驭 AI 辅助工作流，契合团队 Vibe Coding 核心模式

## 优先考虑

- 有 AI 产品、AI Agent 开发经验，热衷并愿意探索 AI 应用前沿
- 有设计系统搭建、实时通信及前端性能优化经验
- 具备服务端开发经验，可参与前后端协同

## 我们的优势

- 深度参与前端 + AI 前沿实践，接触 AI Agent、LLM 集成等热门方向
- 团队开放、鼓励创新，提供清晰的成长路径
- 支持你在前端与 AI 融合领域持续深耕，实现技术与职业双提升
